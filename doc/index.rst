Welcome to Duff crud's documentation!
======================================

This system allow manage clients and their IBANS accounts, the system is based in a docker container architecture.

**Requirements:**

* Docker 19.03.1
* docker-compose 1.24.1
* Sphinx 1.5.6 (only for build documentation)
* sphinx-rtd-theme 0.4.3 (only for build documentation)


**Dependences:**

* Django 2.2.5
* PostgreSQL 11.5
* psycopg2 2.8.3
* djangorestframework 3.9.4
* schwifty 2018.9.1
* social-auth-app-django 3.1.0
* parameterized 0.7.0
